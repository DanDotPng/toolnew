How to use:
1. Drag Package into your Assets folder.
2. Open the tool using the "Tools" option at the top of Unity.
3. Drag prefabs you wish to spawn into the spawnablePrefabs folder and press "Refresh Prefabs" within the tool window.
4. Tick which prefab/prefabs you wish to spawn on the side of the scene view.
5. Select either Random or Single, adjust the radius (either in the tool window or using CTRL + ScrollWheel) and the spawn count.
6. Use CTRL + Space to interact.
7. Use the erase tool to quickly disperse of multiple objects at once.
